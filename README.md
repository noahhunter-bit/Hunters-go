# Hunters-GO 🌍
Website for Noah & Hanna Hunter — Missions and Fundraising Hub

## 🧭 About
A single-page React + Tailwind website serving as an informational and fundraising hub for missions trips, beginning with the Las Vegas GO Trip 2025 through Trinity Bible College.

## ⚙️ Setup
```bash
npm install
npm run dev
```

## 🚀 Build for Deployment
```bash
npm run build
```

## 🌐 Deployment Options
- **Vercel:** push repo to GitHub, then import in [vercel.com](https://vercel.com)
- **Netlify:** connect repo in [netlify.com](https://www.netlify.com)
- **GitHub Pages:** 
  ```bash
  npm run build
  git add dist -f
  git subtree push --prefix dist origin gh-pages
  ```

## 🔗 Domain
Attach your domain `hunters-go.org` via Vercel or Netlify’s free domain settings.
